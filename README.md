# Django Authentication Project

## Overview

This Django project provides a comprehensive authentication system for user management. It supports user registration, login, logout and profile updates, ensuring secure access and management of user accounts.

## Features

- **User Registration:** Allows users to create accounts with email and password.
- **User Login:** Provides secure login functionality for registered users
- **User Profiles:** Enables users to view and update their profile information.
- **Session Management:** Handles user sessions securely.
